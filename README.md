# 🍕 Robo-Saorma — Website

Proiect web realizat în cadrul cursului de **HTML & CSS** la Universitatea Transilvania din Brașov (UNITBV).

## 📁 Structura proiectului

```
Poriect progamare vitor 2/
├── index.html       → Pagina principală (Acasă)
├── meniu.html       → Pagina cu meniul 
├── contact.html     → Pagina de contact și rezervări
├── style.css        → Stilizarea întregului site
└── assets/
    ├── logo.png         → Logo-ul restaurantului
    └── ...
```

## 📄 Pagini

- **Acasă** (`index.html`) — Secțiune hero, avantaje și povestea restaurantului
- **Meniu** (`meniu.html`) — Pizza Clasică și Specialitățile Casei cu prețuri
- **Contact** (`contact.html`) — Adresă, telefon, email, program și hartă

## 🛠️ Tehnologii folosite

- HTML5
- CSS3 

## 🚀 Cum rulezi proiectul

1. Descarcă/clonează repo-ul
2. Deschide `index.html` într-un browser
3. Navighează între pagini din meniu

## 📌 Note

- Site în limba română
- Design responsive cu viewport meta tag
- Imagini externe via Unsplash + imagini locale în `/assets`
